# Security Policy

## Supported Versions

Only the latest tag of each image is supported. Images are tagged by branch (`main`), short commit SHA (`sha-<...>`), semantic version on release (`<image-name>/v*`), and `latest` for the most recent `main` build. Older tags remain accessible but do not receive security updates.

## Reporting a Vulnerability

Report vulnerabilities through GitHub's [private security advisories](https://github.com/ej-east/redoubt/security/advisories/new). Please do ***not*** open a public issue or PR.

I aim to acknowledge reports within 7 days and provide an initial assessment within 30 days. High-severity issues will be prioritized faster.
